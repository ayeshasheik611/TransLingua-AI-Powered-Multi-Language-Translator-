# TransLingua AI Studio

An AI-powered language translation application.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
